# Assignment 13: OOP – About Me & Classmate Guess 🍦

## Objective

* Create and use a class
* Instantiate multiple objects
* Practice methods
* Use GitHub to submit work

---

## 🧁 Your Task 

You will create a program that stores information about:

1. Yourself
2. A classmate (make your best guess!)
3. (Bonus) A third student entered by the user

---

## 💻 Step 1: Create a Class

Create a class called:

Student

Attributes:

* name
* favorite_ice_cream
* favorite_movie
* dream_job

---

## 🔧 Step 2: Add a Method

Create a method:

display_info()

This should print the student’s information clearly.

---

## 🧱 Step 3: Create TWO Objects

In your `main()` function:

### Object 1 (YOU)

Create a Student object using your real information.

---

### Object 2 (CLASSMATE GUESS)

Create another Student object where you:

* Pick someone in class
* Guess their:

Attributes:

* name
* favorite_ice_cream
* favorite_movie
* dream_job

---

## 👉 Display Both

In main, print a label and display both objects:

--- My Info ---
--- My Guess ---

---

## 🎉 BONUS: Add a Third Student (User Input)

Ask the user to enter:

* Name
* Favorite ice cream
* Favorite movie
* Dream job

Create a **third Student object** using this input.

---

### 👉 Display the Third Student

Print:

--- User Entered Student ---

Then call display_info()

---

## 👉 Example Output

--- My Info ---
Hello! My name is Jordan...

--- My Guess ---
Hello! My name is Alex...

--- User Entered Student ---
Hello! My name is Taylor...
